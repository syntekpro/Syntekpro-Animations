## ⚡ QUICK START

### 1. Enable Plugin
Go to **WP Admin → SyntekPro Animations → Settings**  
✅ Enable Animation Engine  
✅ Enable Scroll Animations

### 2. Add Your First Animation

```
[sp_animate type="fadeIn"]Your content here[/sp_animate]
```

### 3. Use Visual Builder (Easiest!)

**WP Admin → SyntekPro Animations → Builder**
1. Select animation
2. Adjust settings
3. Watch live preview
4. Copy code
5. Done! 🎉

## 🎬 ADVANCED FEATURES

### Timeline Sequences
```
[sp_timeline auto="yes"]
    [sp_animate type="fadeIn"]Step 1[/sp_animate]
    [sp_animate type="slideLeft"]Step 2[/sp_animate]
[/sp_timeline]
```

### Staggered Animations
```
[sp_sequence type="fadeInUp" stagger="0.15"]
    <div>Item 1</div>
    <div>Item 2</div>
[/sp_sequence]
```

### Hover Effects
```
[sp_hover_effect type="scale" scale="1.1"]
    <button>Hover Me</button>
[/sp_hover_effect]
```

2. Activate in WordPress
3. Go to **SyntekPro Animations → Settings**
4. Enable Animation Engine
5. Start animating!
