# Storybook Layer Library

This workspace includes Storybook starter stories for built-in slider layer types.

## Included stories

- Badge layer
- Title layer
- Description layer
- Button layer
- Caption layer

## Quick start

1. Add Storybook dependencies in your frontend workspace.
2. Point Storybook to `docs/storybook/stories`.
3. Use the stories as a reference to build custom layer variants.
